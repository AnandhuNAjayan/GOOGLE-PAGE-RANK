This is only an algorithm demonstration project.

This project implements the PageRank algorithm, a link analysis algorithm originally developed by Larry Page and Sergey Brin at Stanford University and later used by Google to rank web pages. The goal of this project is to demonstrate how PageRank computes the importance of nodes in a directed network using probability theory and iterative numerical methods. I inspired to do this by my coursera course "Mathematics for Machine Learning" and also my interest in understaing  the world of linear algebra, calculus, numercial and real analysis.



PROJECT OBJECTIVES ::

The project demonstrates:

A.)Representation of a web graph using an adjacency matrix
B.)Construction of a stochastic transition matrix
C.)Application of the damping factor
D.)Implementation of the PageRank algorithm using power iteration
E.)Convergence analysis of the algorithm
F.)Comparison with the implementation from NetworkX


APPLICATIONS OF PAGE RANK ::

The PageRank algorithm can be applied to many types of networks:

A.)Web page ranking in search engines
B.)Citation analysis in research papers
C.)Social network influence detection
D.)Recommendation systems
E.)Biological network analysis




This project demonstrates the core principles behind the PageRank Algorithm, including stochastic modeling, Markov chains, and iterative convergence methods.
The implementation highlights how network structure influences node importance and provides a practical demonstration of one of the most influential algorithms in web search technology.
